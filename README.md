# Agentic Code Generator

이 프로젝트는 **LangGraph**를 활용해 에이전틱 코드 생성기(Agentic Code Generator)를 구현하는 실습 과정을 담고 있습니다. 코드 생성 과정에서 LangChain과 LangGraph의 다양한 기능을 익히고, 에이전트 기반 코드 작성 자동화를 목표로 합니다.

---

## 🚀 빠른 시작 (Quick Start)

### 🛠️ 환경 설정
- Python 3.11.9 버전 사용 권장

```bash
# Poetry 설치
pip install poetry

# 가상환경 활성화
poetry env activate

# 프로젝트 의존성 설치 및 업데이트
poetry update
```

### ▶️ 실행 방법
```bash
python main.py
```
